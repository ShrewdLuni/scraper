Ceneo Product Review Extractor

Purpose:

The application is designed to extract product reviews from Ceneo.pl using unique product codes. It retrieves and displays product opinions, along with basic statistics and visualizations, helping users analyze customer feedback efficiently.

How It Works:

  1. Backend powered by Python Flask and MongoDB.
  2. Flask handles requests, serving product info from the database or scraping new information as needed.
  3. Product ID input triggers extraction of reviews.
  4. Reviews are analyzed, highlighting pros, cons, and ratings.
  5. Charts are generated, and data can be exported in CSV, XLSX, or JSON formats.

Libraries used:

  • Flask
  • BeautifulSoup (bs4)
  • Requests
  • MongoDB
  • Pandas & Openpyxl
  • Chart.js

Development Approach:

  1. Used Flask for the backend to build a lightweight, flexible app capable of handling requests efficiently.
  2. Implemented MongoDB for its scalability and schema flexibility, allowing dynamic storage and retrieval of product and review data.
  3. Utilized Docker to containerize the application, ensuring consistent development, testing, and production environments.
  4. Employed web scraping techniques within Flask to gather real-time product information when database queries are insufficient.
  5. Focused on data export functionality, providing multiple formats (CSV, XLSX, JSON) for seamless data analysis and sharing.
